# ru-gost-crypto-lab: Лаборатория ГОСТ-криптографии

**ru-gost-crypto-lab** — это лабораторный стенд и набор инструментов для тестирования инфраструктуры с использованием российской криптографии (ГОСТ TLS, S/MIME, VPN).

> [!IMPORTANT]
> Это **лабораторный стенд**, а не инструмент для продакшн-бэкапов.
> Если вам нужен инструмент для защищенных бэкапов с ГОСТ-шифрованием, используйте [gost-backup-vault](https://github.com/ranas-mukminov/gost-backup-vault).

## Назначение

Проект предназначен для:
*   Развертывания тестовых сценариев (Web, Mail, VPN) с поддержкой ГОСТ.
*   Проверки совместимости клиентов и серверов.
*   Нагрузочного тестирования (handshake, throughput).
*   Обучения и экспериментов с конфигурациями (Nginx, HAProxy, Postfix/Dovecot).

## Быстрый старт

### Предварительные требования
*   Docker & Docker Compose
*   Python 3.11+
*   Установленный криптопровайдер (опционально, для локальных тестов)

### Запуск веб-сценария (пример)

```bash
# 1. Клонирование репозитория
git clone https://github.com/ranas-mukminov/ru-gost-crypto-lab.git
cd ru-gost-crypto-lab

# 2. Установка зависимостей
pip install .

# 3. Генерация конфигураций для сценария "Госуслуги"
python3 tools/scenario_generator.py --scenario lab_scenarios/web_portal_gov/scenario.yaml

# 4. Запуск контейнеров
docker-compose -f iac/docker-compose/docker-compose.web.yml up -d
```

### Запуск тестов совместимости

```bash
python3 tools/compatibility/check_tls_clients.py --target localhost:443
```

## Услуги и поддержка – run-as-daemon.ru

Проект развивается инженером DevOps/DevSecOps с сайта [run-as-daemon.ru](https://run-as-daemon.ru).

Если вам нужно:
- протестировать веб, почту, VPN или бэкапы с ГОСТ-криптографией;
- подобрать стек и конфигурацию СКЗИ;
- получить понятные отчёты по совместимости и производительности,

вы можете заказать:
- развёртывание лаборатории под вашу инфраструктуру;
- подготовку отчётов «для руководства» и «для технарей»;
- сопровождение внедрения ГОСТ-инфраструктуры.
