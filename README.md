# ru-gost-crypto-lab

**Laboratory stand for testing infrastructure with GOST cryptography (TLS, S/MIME, VPN).**

> [!NOTE]
> This is a **lab environment**, not a production backup tool. For production backups with GOST encryption, see [gost-backup-vault](https://github.com/ranas-mukminov/gost-backup-vault).

## Overview

This repository provides:
1.  **IaC Scenarios**: Docker/Ansible templates for Web, Mail, and VPN with GOST support.
2.  **Compatibility Tests**: Tools to check which clients can connect to your GOST services.
3.  **Performance Benchmarks**: Scripts to measure handshake times and throughput.
